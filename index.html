Configurar Supabase

Para habilitar la base de datos y actualizaciones en tiempo real, proporciona la URL y la Anon Key de tu proyecto de Supabase.

Instrucciones de Base de Datos

Crea la tabla ejecutando el siguiente SQL en el SQL Editor de Supabase: 

CREATE TABLE social_users (
  id uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  username text NOT NULL,
  platform text NOT NULL CHECK (platform IN ('instagram', 'tiktok')),
  created_at timestamptz DEFAULT now()
);

⚠️ Importante: Recuerda activar el servicio de Realtime en la tabla social_users desde la sección Database → Replication en Supabase para que las tarjetas carguen en tiempo real.
